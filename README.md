## Swift Seminar

[Swift Seminar](http://code0-sh.github.io/SwiftSeminar/)のソースコードです。

ビルドシステムとしてGulp.jsを使用しています。

##### 必要なもの

- Node.js

- npm

##### 使い方

```bash
$ npm install -g gulp
```

```bash
$ npm install
```

```bash
$ gulp
```
